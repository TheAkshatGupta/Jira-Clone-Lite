# 🚀 JIRA Clone Lite

A full-stack task management web application inspired by JIRA, built using React, Node.js, and MongoDB.

---

## 🔥 Features

- 🔐 User Authentication (JWT based)
- 🧑‍💻 User-specific tasks (no data mix)
- ✅ Create, Update, Delete Tasks
- 📊 Task Status Tracking (To-do, In Progress, Done)
- 🎯 Drag & Drop Task Management
- 💻 Responsive Dashboard UI (Tailwind CSS)

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Axios

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## ⚙️ Installation & Setup

**1️⃣ Clone the repository**

git clone https://github.com/TheAkshatGupta/Jira-Clone-Lite.git
cd Jira-Clone-Lite

---

**2️⃣ Backend Setup**

cd server
npm install
node index.js

---

**3️⃣ Frontend Setup**

cd client
npm install
npm start

---

## 🔐 Environment Variables

Create a ".env" file inside the "server" folder and add:

MONGO_URI=your_mongodb_connection_string
